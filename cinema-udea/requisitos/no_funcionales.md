
# Requisitos No Funcionales

- RNF1. Usabilidad: interfaz de consola clara con menús y mensajes descriptivos.
- RNF2. Fiabilidad: validaciones de entrada y manejo de errores básicos.
- RNF3. Compatibilidad: Python 3.10+; ejecución en Google Colab.
- RNF4. Rendimiento: operaciones en memoria para una sala de 121 asientos.
- RNF5. Seguridad básica: acceso al módulo administrador con usuario/clave.
