
# Requisitos Funcionales

- RF1. Registrar usuarios (validaciones: nombre/apellido ≥3 letras, sin números; documento 3–15 dígitos, solo números; vínculo permitido).
- RF2. Crear reservas (1 asiento por reserva), selección de función y asiento; actualizar asientos "O"→"X".
- RF3. Cancelar reservas; asiento "X"→"O", validación de reservas activas.
- RF4. Consultar funciones del **próximo fin de semana** (día, hora, nombre, sala, sillas disponibles).
- RF5. Imprimir factura en consola para cada reserva.
- RF6. Módulo de administrador con: totales (reservas/tiquetes/pagos), promedio diario, lista de usuarios, mayor/menor reservas.
- RF7. Exportar resultados a **CSV**.
